# TailwindCSS Ghost Blog

Inspired by nothing in particular. A dead simple ghost blog. 


## Installation

```
cd ..
zip spcss-theme.zip spcss-ghost -r -x 'node_modules/' 
cd spcss-ghost
```

Then upload this to your Ghost site.
